# Events Ticketing
Small example event ticket booking app built with:

![App screenshot](/app-screenshot.png?raw=true "Event Ticketing App")

### Backend
- NestJS
- PostgreSQL & TypeORM
- Elasticsearch

### Frontend
- NextJS

## How to run
In root directory:
```docker-compose up```

Client runs on ```localhost:3006```, Server runs on ```localhost:3000```
